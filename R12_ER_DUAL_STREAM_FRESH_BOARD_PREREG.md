## 1. Purpose

The old-family train-only result establishes that cardinality-marginalized
monotone routing can learn exact variable-relation transport. It does not test
fresh surface families. This board is the first score-bearing falsifier. It
changes the namespace, grammar, renderer compositions, physical record order,
semantic families, and adds irrelevant opaque tokens while preserving the
same public finite-relation task.

## 3. Neutral namespace and distractors

Every entity, witness, opcode, and irrelevant token has the same six-byte
shape `z[0-9a-z]{5}`. There is no entity/opcode/witness prefix category.

The candidate contract is exact:

- declaration: `2N` semantic candidates;
- active rule: `2N+1` candidates, with one opcode at a renderer-dependent
  ordinal and `2N` ordered witnesses;
- inactive rule: one irrelevant candidate;
- event tape: twelve semantic opcode occurrences plus exactly one irrelevant
  candidate across thirteen records and one explicit `END`;
- late query: one irrelevant candidate plus the requested numeric position.

The distractors never alias a semantic symbol. Rotating every distractor
within a row must preserve public-byte parser output, recurrent state, and
answer exactly. Candidate counts, not hidden metadata, define the ordinal
lattice's admissible paths.

## 4. Renderer split
